How to Contribute to Any Project

Follow these standard steps to make your first contribution to any open-source repository.

1. Find an Issue & Read Guidelines

Before writing code, check the project's Issues tab for something to fix. Always read CONTRIBUTING.md to understand the project's rules.

2. Fork the Repository

Click the Fork button on the top right of the repository page. This creates your own copy of the project where you can make changes safely.

3. Clone Your Fork

Download your forked copy to your local machine using the terminal/command prompt.

git clone https://github.com/YOUR-USERNAME/PROJECT-NAME.git

4. Create a New Branch

Never work on the main branch. Create a feature branch with a descriptive name.

git checkout -b feature/add-new-guide

5. Make Changes & Commit

Edit the files in your code editor. Once done, stage and commit your changes with a clear message.

git add . git commit -m "Added a detailed guide on Git workflow"

6. Push Changes

Upload your changes from your computer to your forked repository on GitHub.

git push origin feature/add-new-guide
