Open Source Programs

Explore popular open source programs that help students and developers gain real-world experience through contributions.

Social Winter of Code

A beginner-friendly winter program focused on open source contribution with mentorship from experienced developers.

GirlScript Summer of Code

A global open source program encouraging students to contribute to real-world projects during the summer.

Google Summer of Code

A prestigious international program that pairs students with open source organizations for paid internships.

Outreachy

A paid internship program focused on supporting underrepresented groups in open source and tech.

MLH Fellowship

A remote program that helps students build open source projects while working closely with mentors.

Linux Foundation Mentorship

Professional-level mentorship programs for contributors working on large-scale open source ecosystems.
