How It Works

Welcome to Git Playground!

This tool is designed to help you practice Git commands in a safe, simulated environment. You can type commands in the terminal on the left and see the results instantly, without worrying about breaking anything.

Interface Guide

  • Terminal: The main area where you type and execute Git commands.
  • Cheat Sheet: Click on any command in the right panel to see what it does.
  • Status Bar: Shows your current user (user@compass) and branch.

How to Use

  • Type git init to start a new project.
  • Use git status to check for changes.
  • Type git add . to stage all your files.
  • Type git commit -m "message" to save changes.

Handy Tips

  • Press the Up Arrow key to repeat your last command.
  • Type clear to wipe the terminal screen clean.
  • Check Settings to change the theme or font size.
