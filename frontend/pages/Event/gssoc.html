About GirlScript Summer of Code

GirlScript Summer of Code (GSSoC) is an initiative to empower women developers by providing mentorship, real-world open-source projects, and a platform to enhance coding skills while contributing to impactful software. Join us to learn, code, and make a difference in the tech community.
