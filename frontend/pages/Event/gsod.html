Google Season of Docs

An initiative by Google connecting professional technical writers with open source communities to improve documentation quality.

Visit Official Website →

Program Overview

Documentation Focus

Google Season of Docs helps open source organizations improve technical documentation, onboarding guides, API references, and overall developer experience.

Writer Collaboration

Professional technical writers collaborate directly with open source maintainers to identify documentation gaps, structure content, and improve clarity.

Funded Grants

Participating organizations receive financial grants from Google to support documentation projects and compensate technical writers.

Project Growth

Well-documented projects attract more contributors, improve usability, and increase long-term sustainability within the open source ecosystem.

Who Can Apply?

Technical Writers

Experienced writers with documentation expertise.
