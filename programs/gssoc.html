GirlScript Summer of Code (GSSoC)

Annual

GirlScript Summer of Code is a three-month-long open-source program conducted by the GirlScript Foundation. It is designed to help beginners get started with Open Source Development.

Key Details

  • Season: Summer (May - Aug)
  • Duration: 3 Months
  • Eligibility: Students & Professionals
